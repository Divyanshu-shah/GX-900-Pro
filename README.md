📁 Project Structure
======================
The project is organized into the following folders:
* **components**: This folder contains the application's React components, including UI elements and layout components.
* **pages**: This folder contains the application's pages, including the homepage and any other routes.
* **public**: This folder contains the application's static assets, including images and fonts.
* **styles**: This folder contains the application's CSS styles, including the global CSS file and any component-specific styles.
* **utils**: This folder contains the application's utility functions, including API helpers and data formatting functions.
* **next.config.ts**: This file contains the application's Next.js configuration, including settings for server-side rendering and static site generation.

⚙️ How to Run
================
To run the application, follow these steps:
1. **Install dependencies**: Run `npm install` or `yarn install` to install the application's dependencies.
2. **Start the development server**: Run `npm run dev` or `yarn dev` to start the development server.
3. **Build the application**: Run `npm run build` or `yarn build` to build the application for production.
4. **Start the production server**: Run `npm run start` or `yarn start` to start the production server.

**Environment Variables**:
* **NODE_ENV**: Set to `development` or `production` to control the application's environment.
* **PORT**: Set to a custom port number to override the default port.

**Setup**:
* **Create a new Next.js project**: Run `npx create-next-app my-app` to create a new Next.js project.
* **Install dependencies**: Run `npm install` or `yarn install` to install the application's dependencies.
* **Configure the application**: Update the `next.config.ts` file to configure the application's settings.
